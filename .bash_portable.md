# Bash Portable Configuration

## Pliki do transferu na inne maszyny

1. `~/.bashrc` - główny plik konfiguracyjny (tylko dodaj linie sourcujące)
2. `~/.bash_prompt` - konfiguracja prompta z systemem aliasów hostów
3. `~/.bash_aliases` - aliasy użytkownika (zawiera migrację standardowych aliasów z .bashrc)
4. `~/.bash_functions` - custom funkcje (wymaga `fzf`)
5. `~/.inputrc` - konfiguracja klawiszy funkcyjnych (wymaga `fzf`)

## Modyfikacje w .bashrc

Dodaj na końcu pliku `.bashrc` następujące linie:

```bash
# Custom bash configuration
if [ -f ~/.bash_prompt ]; then
    . ~/.bash_prompt
fi

if [ -f ~/.bash_functions ]; then
    . ~/.bash_functions
fi
````

Plik `.bash_aliases` jest automatycznie ładowany przez bash jeśli istnieje.

## Konfiguracja aliasów hostów

W pliku `~/.bash_prompt` znajduje się tablica `HOSTNAME_ALIASES`:

```bash
HOSTNAME_ALIASES=(
    "wawsrvdev175:dev"
    # dodaj więcej hostów tutaj
)
```

System obsługuje tylko mapowanie short-short (krótki hostname do krótkiego aliasu).
Na przykład: `wawsrvdev175` → `dev`

## Użycie

1. Skopiuj pliki na nową maszynę
2. Dodaj odpowiednie wpisy do `/etc/hosts` (aliasy)
3. Zmodyfikuj tablicę `HOSTNAME_ALIASES` w `~/.bash_prompt` jeśli potrzeba
4. Uruchom nową sesję bash lub wykonaj `source ~/.bashrc`

## Użycie klawiszy

* `Ctrl+Page Up` - zmiana katalogu na katalog rodzica (`cd ..`)
* `Ctrl+Page Down` - interaktywny wybór podkatalogu z użyciem `fzf`

## Wymagania

* `fzf` - narzędzie do interaktywnego wyszukiwania (wymagane dla klawiszy Ctrl+Page Down)
